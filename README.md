# logging

This library wraps the logging methods of the console object to provide basic LOG_LEVEL functionality. We use CodeArtifact as a private npm registry for the @digital-plant namespace.

`import '@digital-plant/logging'`

This is a private npm package for use by Digital Plant code.

It is hosted by Digital Plant's private CodeArtifact store.

If you make changes, then update the version in the package.json and perform `npm publish`.
